ព័ត៌មានវត្តអារាម

ព័ត៌មានទាំងអស់អំពីវត្តសាលាគូរ (វត្តអំពិលបី)

ព័ត៌មានមូលដ្ឋាន

ប្រវត្តិវត្ត

វត្តសាលាគូរ ឬហៅថា វត្តអំពិលបី ត្រូវបានសាងសង់ឡើងនៅឆ្នាំ ១៩៩៨។ វត្តនេះត្រូវបានកសាងដោយព្រះអង្គម្ចាស់សម្តេចព្រះមហាសុមេធាធិបតី។ វត្តនេះជាកន្លែងសក្ការៈបូជា និងកន្លែងសិក្សាធម៌សម្រាប់ពុទ្ធបរិស័ទក្នុងតំបន់។

ទីតាំង

វត្តសាលាគូរ ស្ថិតនៅភូមិម្លូមុឺន ឃុំផ្សារដែក ស្រុកពញាឮ ខេត្តកណ្ដាល។ ចម្ងាយប្រហែល ១៥គីឡូម៉ែត្រពីទីរួមខេត្ត។ វត្តមានទីតាំងស្ថិតនៅក្បែរផ្លូវជាតិលេខ៥។

សំណង់សំខាន់ៗ

វត្តមានសំណង់សំខាន់ៗដូចជា៖ - ព្រះវិហារធំ (៣០ម៉ែត្រ x ២០ម៉ែត្រ) - សាលាបាលី - កុដិព្រះសង្ឃ ១៥កុដិ - បរិវេណសួនច្បារធំ

ប្រវត្តិសង្ខេប

១៩៩៨

ការបង្កើតវត្ត

វត្តត្រូវបានបង្កើតឡើងដោយព្រះអង្គម្ចាស់សម្តេចព្រះមហាសុមេធាធិបតី នៅលើទីដីដែលគ្រួសារម្ចាស់ផ្ទះអោយប្រគេន។

២០០៥

ការសាងសង់ព្រះវិហារ

ព្រះវិហារធំត្រូវបានសាងសង់ឡើងជាលើកដំបូង ដោយមានការឧបត្ថម្ភពីពុទ្ធបរិស័ទក្នុងតំបន់។

២០១០

ការបើកសាលាបាលី

សាលាបាលីត្រូវបានបើកសម្រាប់ការសិក្សាព្រះត្រៃបិដក និងការអប់រំភាសាខ្មែរដល់កុមារក្នុងសហគមន៍។

២០១៥

ការជ្រើសរើសព្រះចៅអធិការថ្មី

ព្រះតេជគុណ ធម្មវិរិយោ ត្រូវបានតែងតាំងជាព្រះចៅអធិការវត្ត បន្ទាប់ពីការចូលព្រះនិព្វានរបស់ព្រះចៅអធិការមុន។

២០២៤

វត្តបច្ចុប្បន្ន

វត្តមានសកម្មភាពសង្គម អប់រំ និងសាសនាជាច្រើន ដែលជួយដល់សហគមន៍ក្នុងតំបន់។

បរិក្ខារនិងសេវាកម្ម

កន្លែងស្នាក់នៅ

- កុដិព្រះសង្ឃ ១៥កុដិ
- បន្ទប់សម្រាប់ភ្ញៀវ (២បន្ទប់)
- បន្ទប់សិក្សាធម៌
- បន្ទប់ប្រជុំ

បរិភោគាហារ

- ភោជនីយដ្ឋានសម្រាប់ព្រះសង្ឃ
- បន្ទប់ធ្វើម្ហូបសម្រាប់ពុទ្ធបរិស័ទ
- កន្លែងអង្គុយសម្រាក
- ទឹកស្អាតគ្រប់ពេល

ការសិក្សា

- បណ្ណាល័យព្រះត្រៃបិដក
- សាលាបាលី
- ថ្នាក់រៀនភាសាខ្មែរ
- ថ្នាក់រៀនសមាធិ

ច្បាប់និងបទបញ្ជា

ច្បាប់សម្រាប់ភ្ញៀវមកទស្សនា៖

  • ✓ ត្រូវពាក់សម្លៀកបំពាក់សមរម្យ
  • ✓ ត្រូវដោះស្បែកជើងមុនចូលព្រះវិហារ
  • ✓ មិនត្រូវនិយាយសំឡេងខ្លាំងក្នុងវត្ត
  • ✓ មិនត្រូវថតរូបដោយគ្មានការអនុញ្ញាត
  • ✓ ត្រូវគោរពព្រះសង្ឃនិងកន្លែងសក្ការៈ
